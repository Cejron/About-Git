# About-Git
Git使用方法以及常用命令

 ### 1.下载Git [https://git-scm.com/download/win](https://git-scm.com/download/win)
 
得到快捷方式Git Bash

### 2.从本地创建仓库到把本地代码上传到版本库

1.文件夹+Git Bash Here 打开控制台

2.ls -a 查看当前文件目录

3.git init 初始化一个空的git仓库(文件目录下会多一个隐藏的.git文件)

4.git status 查看文件状态(有没有被追踪)

四种状态

  >Untracked
  
  >Modified
  
  >Staged
  
  >Committed
